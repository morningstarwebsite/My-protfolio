# Chiamaka Blessing | Backend Developer Portfolio

A modern, animated portfolio website showcasing my work as a Backend Developer specializing in Node.js, Express.js, and PostgreSQL. Built with vanilla HTML, CSS, and JavaScript featuring interactive animations, smooth scrolling, and a fully functional contact form.

## 🌟 Features

- **Animated Starfield Background** – Canvas-based particle system with twinkling stars
- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile
- **Smooth Scroll Navigation** – Active link tracking based on scroll position
- **Project Showcase** – Featured and secondary projects with tech stack details
- **Skills Grid** – Categorized technical skills with icons
- **Contact Form** – Integrated with Formspree for email submissions
- **Dynamic Footer** – Auto-updating copyright year
- **Hover Effects** – 3D rotation on skill cards and parallax on project images
- **Mobile Menu** – Toggle navigation for smaller screens

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3 (Grid, Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (ES6+)
- Canvas API (Starfield animation)

**Backend Integration:**
- Formspree (Contact form submissions)
- SendGrid (Email services)
- Africa's Talking (SMS integration)

**Deployment:**
- Render (Live demo hosting)

## 🚀 Getting Started

### Prerequisites
- Node.js and npm (for local development server)
- A modern web browser

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/morningstarwebsite/My-protfolio.git
   cd My-protfolio
   ```

2. **Start a local development server:**
   ```bash
   npx serve .
   ```
   The portfolio will be available at `http://localhost:3000` (or the port specified in terminal)

3. **Or open directly in browser:**
   Simply open `index.html` in your web browser to view locally without a server
